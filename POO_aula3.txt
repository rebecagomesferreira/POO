tipos de atributos de classe:
private

controle de atributos:
encapsulamentos

controle de acesso:
getters e setters

herança: classe herda características e comportamentos de outras classes
ex: a subclasse moto herdaria características da classe veículo + suas proprias 
public class moto extends veiculo

polimorfismo: habiliades de um objetos responder a mesma mensagem de forma diferente
ex: public class ligar() - ligado = True
public class ligar(Mobile m) - ligado = m.get.True - liga pelo  celular

abstração: conceitos do mundo real através de classes e objetos, focando nas caracteríscas essenciais

Classe: entidades 

Construtures: metodos de inicialização, metodos que possuem o mesmo nome da classe, após finalizada é usada como modelo do objeto

Objeto: cada objeto criado a partir de uma classe é uma instância, representa a entidade do mundo real
ex: classe gato
